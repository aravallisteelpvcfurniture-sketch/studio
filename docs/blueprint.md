# **App Name**: aravallisteel

## Core Features:

- Immersive Product Showcase: Engaging hero slider with high-definition product imagery and category-based browsing through vertically snapping, large rectangular cards with rounded corners for modular kitchens, wardrobes, and other interior solutions.
- Detailed Product View: Dedicated screen to explore specific designs, featuring a large main image, floating info panel with material details, color options (small circles), and warranty information.
- Instant Budget Estimator: An interactive utility tool allowing users to select area (Square Feet) and material type via dropdowns to receive a live, tentative budget estimate, with the option to download the estimate as a PDF.
- AI-Powered Design Idea Tool: A generative AI tool providing personalized design tips and conceptual ideas based on user input regarding their space, style preferences, or current trends.
- Direct Engagement & Support: Streamlined features for user inquiries, including 'Get Quote' and 'Enquire on WhatsApp' buttons, alongside direct contact options for support and an 'About Us' section with company history and certifications.
- Discovery & Experience Tools: Facilitate user exploration with features like a dealer locator, integrated 360° virtual tours of select designs, and a gallery showcasing real customer installations.

## Style Guidelines:

- Primary color: Sophisticated Dark Grey (#4A4A4D), evocative of metallic finishes and professional quality.
- Background color: A subtle, cool-toned near-white (#F7F7F8), ensuring a clean and minimalist canvas.
- Accent color: Vibrant Cyan Blue (#26B2D9), providing a modern pop and highlighting key interactive elements.
- Primary font: 'Roboto' (sans-serif) for both headlines and body text, ensuring a clean, modern, and highly readable aesthetic. Note: currently only Google Fonts are supported.
- Minimalist and uncluttered screen layouts, prioritizing 'no long scroll'. Utilize vertical snapping for category exploration and large, rounded-corner cards for product display. Main product images occupy 60% of the screen with information provided in a floating panel.
- Use clean, minimalist line icons for navigation and quick action buttons to complement the premium, modern aesthetic.
- Implement auto-scrolling for the hero slider and subtle, smooth transitions for UI elements and screen changes to enhance the 'VIP Look' without being distracting.
- Main navigation housed within a simple '3-Dot' menu (hamburger menu) to keep the primary UI minimalist.
